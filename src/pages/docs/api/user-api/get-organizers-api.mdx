# Get Organizers

## `getOrganizers`

### Description

Fetches all users with the role of "organizer".

### Parameters

None.

### Returns

- A JSON array of users with the role of "organizer".

### Example

```javascript
import { getOrganizers } from "path/to/api";

getOrganizers()
  .then((organizers) => console.log(organizers))
  .catch((error) => console.error(error));
```

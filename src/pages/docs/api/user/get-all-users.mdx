# Get All users

## `getAllUsers`

### Description

Fetches all users from the database.

### Parameters

None.

### Returns

- A JSON array of all users in the database.

### Example

```javascript
import { getAllUsers } from "path/to/api";

getAllUsers()
  .then((users) => console.log(users))
  .catch((error) => console.error(error));
```

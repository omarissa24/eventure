# Getting Started

Welcome to the documentation for our Eventure application. This guide will help you get started with running the development server and accessing the production version of the website.

## To Run the Development Server

Follow the steps below to start the development server:

1. Open your terminal.
2. Navigate to the project directory.
3. Run the following command:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## To Open the Website in Production

You can access the production version of the website using the link below:

[https://eventure-rho.vercel.app/](https://eventure-rho.vercel.app/)

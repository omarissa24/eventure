# Getting Started

Welcome to the documentation for our Eventure application. This guide will help you get started with running the development server and accessing the production version of the website.

## Installation Guide

Before running the development server, ensure you have Node.js and npm installed. Follow the steps below to install them:

1. **Install Node.js and npm:**

   - Visit the [Node.js website](https://nodejs.org/) and download the latest stable version.
   - Follow the installation instructions for your operating system.

2. **Verify Installation:**

   - Open your terminal and run the following commands to check if Node.js and npm are installed correctly:

     ```bash
     node -v
     npm -v
     ```

   - You should see the versions of Node.js and npm printed in the terminal.

## To Run the Development Server

Follow the steps below to start the development server:

1. **Clone the [Repository](https://github.com/omarissa24/eventure):**

   ```bash
   git clone https://github.com/omarissa24/eventure.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd eventure
   ```

3. **Install the Necessary Dependencies:**

   ```bash
   npm install
   ```

4. **Start the Development Server:**

   ```bash
   npm run dev
   ```

5. **Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.**
